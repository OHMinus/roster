How to use

1. ディレクトリ選択

入団届があるpdfがまとめられているディレクトリを選択してください。

2. 確認すべき団員を確認

このページの左側のカラムから赤字になっている団員は入団届に不備がある団員です。

確認の後、変更や再提出を要求してください。

3. 出力する

残念ながら、セッションセーブの機能はありません。

現在のセーブは以下の通りです。

セーブ方法 詳細
Save UTF-8で保存します。
Google spread sheetで開く時はこちらを選んでください。
Save (compatible with Excel) Shift-jisで保存。
Microsoft 365のアプリ(Excel,Access等)で開かない限りはこちらを選ぶ必要はありません。
